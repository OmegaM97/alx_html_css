# ALX School Web Page

This is a responsive landing page built using HTML and CSS. It showcases ALX School with sections like banner, testimonials, videos, membership details, and FAQs.

## Features

- Responsive design using Flexbox
- Header with logo and navigation links
- Hero section with call to action
- Testimonials and featured professionals
- Video tutorials and membership info
- FAQ and footer with social media links

## Technologies

- HTML5
- CSS3
- Font Awesome (CDN)

## Usage

Open `index.html` in your browser to view the page. Make sure the `photos/` folder is present and contains all the images referenced in the file.
